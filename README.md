# Exceed Portal

Login dashboard for Exceed Real Estate (Dubai).

- Agent and Client login entry points
- Agent self-registration restricted to `@exceed-re.ae` company emails
- English / Japanese language toggle (defaults to Japanese)
- Demo backend uses `localStorage` (swap for real API when ready)

## Run locally

Open `index.html` in a browser, or serve the folder:

```bash
python3 -m http.server 8000
```
